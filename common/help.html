Direct Currency Converter

Getting started

Welcome to Direct Currency Converter (DCC) 3.4.0.

The conversion of prices on a newly opened browser tab will be done only when you press the $€ icon (it turns green).

DCC can get its currency quotes from two different data sources.

  • ECB (33 currencies, default)
  • Currencylayer (168 currencies, requires a free or paid API key)

The data source is selected in the DCC options.
The easiest way is to use ECB.
If you need more currencies, Currencylayer is recommended.
You need an API key that is available from their site.
Store the key in the DCC settings, and you are ready to go.
